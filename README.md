# DittoGolf
This app utilizes pose estimation to provide a side-by-side or overlayed comparison of the user's swing with their ideal swing.

## Tech Stack
- **React**
- **Expo**
- **TypeScript**
- **NativeWind** (Tailwind for RN)

## Project Setup

### Running
Run `docker compose up`

Frontend exposed at `localhost:3000`
Backend exposed at `localhost:8000`
Postgres exposed on `localhost:5432`
